# G3D Import Blender Addon

Плагин для Blender, предназначенный для импорта 3D-моделей формата `.G3D`. Особенностью данного аддона является алгоритм **Deep Scan** (глубокое сканирование структуры файла) и **Infinity Protection** (фильтрация некорректных координат), что позволяет успешно извлекать меши даже из поврежденных или нестандартных файлов.

## Основные возможности

*   **Deep Scan**: Автоматический поиск структур вершин и индексов в бинарных данных.
*   **Infinity Protection**: Предотвращение «разлета» геометрии в бесконечность (фильтрация мусорных координат).
*   **Поддержка различных структур (Strides)**: Обработка данных с шагом 16, 20, 24, 32, 40, 44, 48 и 52 байта.
*   **Автоматический поиск текстур**: Сканирует папку с моделью на наличие подходящих изображений (`.bmp`, `.png`, `.jpg`, `.tga`, `.dds`) и автоматически настраивает материалы (Principled BSDF).
*   **Настройка лимитов**: Возможность задать порог фильтрации координат прямо в окне импорта.

## Установка

1.  Скачайте файл `__init__.py`.
2.  В Blender перейдите в `Edit > Preferences > Add-ons`.
3.  Нажмите `Install...` и выберите файл (или упакованный `.zip` архив с этим файлом).
4.  Активируйте аддон **"G3D Import Blender Addon"**.

## Использование

1.  Перейдите в меню `File > Import > G3D (.G3D)`.
2.  Выберите нужный файл.
3.  В правой панели настроек можно изменить параметр **Infinity Limit** (по умолчанию 20 000), если модель обрезается или, наоборот, содержит слишком много «мусора».
4.  Нажмите **Import G3D (Safe)**.

## Ограничения

*   Аддон использует эвристические методы для поиска данных, поэтому в некоторых случаях может потребоваться ручная корректировка расположения объектов.
*   Импорт ориентирован на извлечение геометрии и базовых материалов; поддержка сложной анимации (Rigging) в текущей версии ограничена.
